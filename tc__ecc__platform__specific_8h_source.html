62  * The RNG function should fill 'size' random bytes into 'dest'. It should
63  * return 1 if 'dest' was filled with random data, or 0 if the random data could
64  * not be generated. The filled-in values should be either truly random, or from
65  * a cryptographically-secure PRNG.
66  *
67  * A cryptographically-secure PRNG function must be set (using uECC_set_rng())
68  * before calling uECC_make_key() or uECC_sign().
69  *
70  * Setting a cryptographically-secure PRNG function improves the resistance to
71  * side-channel attacks for uECC_shared_secret().
72  *
73  * A correct PRNG function is set by default (default_RNG_defined = 1) and works
74  * for some platforms, such as Unix and Linux. For other platforms, you may need
75  * to provide another PRNG function.
