bool ndn_timer_fire_before ( ndn_timer_t *  lhs,
ndn_timer_t *  rhs,
uint64_t  now 
)

This method indicates if the fire time of lhs timer is strictly before the fire time of the rhs timer.

Parameters
lhs.Input. Left-hand-side timer.
rhs.Input. Right-hand-side timer.
now.Input. The current time (may in milliseconds or microsecond).
Returns
TRUE If the fire time of lhs timer is strictly before rhs timer's fire time
FALSE If the fire time of lhs timer is the same or after rhs timer's fire time.
