int tc_aes128_set_decrypt_key ( TCAesKeySched_t  s,
const uint8_t *  k 
)

Set the AES-128 decryption key Uses key k to initialize s.

Returns
returns TC_CRYPTO_SUCCESS (1) returns TC_CRYPTO_FAIL (0) if: s == NULL or k == NULL
Note
This is the implementation of the straightforward inverse cipher using the cipher documented in FIPS-197 figure 12, not the equivalent inverse cipher presented in Figure 15
Warning
This routine skips the additional steps required for keys larger than 128, and must not be used for AES-192 or AES-256 key schedule – see FIPS 197 for details
Parameters
sIN/OUT – initialized struct tc_aes_key_sched_struct
kIN – points to the AES key
